*ReṼoman* is an API orchestration/automation tool for JVM (Java/Kotlin) from the API-first SaaS company, *Salesforce*.
It re-imagines API orchestration/automation by letting you execute a Postman collection in a JVM program/test.

NOTE: NO licensed Postman SDKs are used inside this project. This is written ground up natively in JVM.

== 📖 Documentation

**https://salesforce-misc.github.io/ReVoman/**

== Artifact

Maven
[source,xml,subs=attributes+,tabsize=2]
----
<dependency>
    <groupId>com.salesforce.revoman</groupId>
    <artifactId>revoman</artifactId>
    <version>{revoman-version}</version>
</dependency>
----

Gradle Kts
[source,kts,subs=attributes+]
----
implementation("com.salesforce.revoman:revoman:{revoman-version}")
----

Minimum Java Version required = *21*
