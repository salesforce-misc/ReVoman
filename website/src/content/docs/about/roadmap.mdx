description: Future features planned for ReVoman
---

## Completed

- **JetBrains HTTP Client support** — `.http` file parsing with response handlers (`> {% %}`) and pre-request handlers (`< {% %}`), `http-client.env.json` environment files, and the HTTP Client JavaScript API (`client.global`, `response.body`, etc.)

## Planned

- **API Metrics and Analytics**
- **Support more template formats**
  - Support formats like Cucumber to take input data and assert output data
  - Support formats for other popular tools, such as Bruno, ThunderClient, Insomnia, etc.
- **Payload generation**
- **Flow control through YAML config**
